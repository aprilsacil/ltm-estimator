# LifeTrackMed Technical Exam

## Installation
1. clone the repository `https://github.com/aprilsacil/ltm-estimator.git`
2. run `composer install` inside the project folder
3. run `yarn build` inside the project folder
4. to start the server you can:
    * run the following command inside the project folder:
        `bin/cradle server -h 127.0.0.1 -p 8888`
      and then Open your browser to http://127.0.0.1:8888 and you should see the application running.
